# Game Station

Game Station is a web-based gaming platform that brings together multiple classic browser games in a single, interactive website. The project demonstrates front-end web development skills through responsive design, game logic implementation, and DOM manipulation using JavaScript.

## Features

* Multiple games in one platform

  * Rock Paper Scissors
  * Connect Four
  * Snake Game
* Interactive and responsive user interface
* Real-time game logic using JavaScript
* User-friendly design with smooth navigation between games

## Games Included

### Rock Paper Scissors

A classic game where the player competes against the computer. The game randomly generates the computer's choice and determines the winner based on standard rules.

### Connect Four

A two-player strategy game where players take turns dropping colored pieces into a grid, aiming to connect four pieces horizontally, vertically, or diagonally.

### Snake Game

Control the snake to collect food while avoiding collisions with the walls or its own body. The score increases as the snake grows longer.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/game-station.git
```

2. Open the project folder.

3. Launch **Game-station.html** in your web browser.

No installation or additional dependencies are required.
